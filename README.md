
#  GALAGA – Arcade Shooter Fan Project

A modern, visually immersive fan-made tribute to the legendary 1981 arcade shooter **Galaga**.

This project recreates the retro arcade vibe with a cinematic video background, interactive UI, merchandise shop, and playable arcade access.

---

##  Live Features

*  Fullscreen gameplay video background
*  Pixel-style arcade UI design
*  Playable Galaga (external arcade link)
*  Merchandise Shop section
*  History page about Galaga’s legacy
*  Smooth hover animations & glowing effects
*  Responsive layout (mobile friendly)

---

##  Pages Overview

###  Home Page

* Hero slider with gaming news
* Enemy showcase sections
* Arcade machine quick-play panel

###  History Page

* Legacy and cultural impact of Galaga
* Styled content section with blur-glass effect

###  Shop Page

* Product cards with hover effects
* T-Shirt & Keychain items

###  Game Page

* Clickable arcade machine
* Opens playable Galaga externally

---

* HTML5

* Google Fonts (Press Start 2P, Roboto)

##  Key Design Concepts

* Fixed fullscreen video background
